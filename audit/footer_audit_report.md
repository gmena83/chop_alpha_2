# CHOP ETA Project: Footer Structure Audit Report

## Introduction
This report details a pixel-perfect comparison of the footer structure between the Figma blueprint and the live website for the CHOP ETA project. The audit focuses on footer layout, navigation columns, links, branding, and styling.

## Methodology
Both the Figma design file (https://www.figma.com/design/JblMdPD33uMVKIQ6zAs3e9/GMF-CHOP-ETA-Prototype) and the live website (https://chopeta.xyz/) were inspected. Screenshots were captured for visual comparison, and specific elements were analyzed for discrepancies in measurements, hex values, and pixel dimensions.
